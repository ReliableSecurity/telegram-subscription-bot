Как добавить бота в группу/канал

Для групп:

  1. Откройте группу в Telegram
  2. Нажмите на название группы вверху
  3. Выберите "Добавить участников"
  4. Найдите бота по имени: @jnhghyjuiokmuhbgbot
  5. Добавьте бота в группу
  6. Сделайте бота администратором с правами:
    • Удаление сообщений
    • Блокировка пользователей
    • Изменение информации
    • Закрепление сообщений

Для каналов:

  1. Откройте канал в Telegram
  2. Нажмите на название канала вверху
  3. Выберите "Управление"
  4. Нажмите "Администраторы"
  5. Добавьте бота: @jnhghyjuiokmuhbgbot
  6. Дайте боту необходимые права администратора

Получение ID группы/канала:

  1. Добавьте бота в группу/канал
  2. Напишите в группе/канале: /id
  3. Бот покажет ID группы/канала
  4. Скопируйте ID и вставьте в форму выше
